# Inventory-Tracker-
This is a simple inventory tracking application thats lets you add items, view items and delete items.
The project has a two tier architecture with a backend and frontend.
The frontend UI uses HTML,CSS,Javascript
The backend API uses python
The database uses PostgreSQL
The containerization uses docker
